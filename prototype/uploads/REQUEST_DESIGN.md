## 1. 디자인 시스템 (변경 없음 — 유지)

기본 시안의 다음 요소는 그대로 유지합니다.

- **컬러**: 다크 사이드바 + 라이트 본문, Black/Blue/Red 3원색
- **폰트**: Pretendard
- **타이포**: tabular-nums(숫자 가독성)
- **상태 컬러**: 완료=Green, 예정=Blue, 미정=Amber, 지연=Red
- **반응형 브레이크**: 980px(모바일), 540px(소형 모바일)
- **차트**: Chart.js 기반

---

## 2. 사업부 계층 구조 반영 (필수)

기본 시안은 4개 사업부를 평면(flat)으로 두었으나, 실제는 부모-자식 구조입니다.

```
IT사업부 (모회사 / 통합 보고 단위)
 ├─ 처음서비스 (FS)
 ├─ 마이메일러 (MM)
 └─ 프링고 (PG)
```

### 사이드바 변경
- IT사업부를 펼쳐지는 부모 메뉴로 변경
- 옆에 "통합" 작은 뱃지 추가
- 자식 3개는 들여쓰기, 사업부 컬러 도트 유지
- IT사업부 자체 클릭 → 통합 대시보드, chevron 클릭 → 자식 토글

### IT사업부 통합 대시보드 (신규)
- 페이지 제목: **"IT사업부 통합 대시보드"**
- 서브타이틀: "처음서비스 · 마이메일러 · 프링고 통합 합산"
- KPI 라벨에 "통합" 단어 자동 삽입 (예: "당월 통합 예정 수금")
- **자회사별 매출 기여도 카드** (그룹 막대 차트, 클릭 시 해당 자회사로 이동)
- **자회사별 요약 카드** (달성률 바 + 실제/예정 금액 + 건수)

## 3. UI/UX 수정 요청 사항

### 3.1 대시보드 공통

| # | 항목 | 현재 | 수정 요청 |
|---|---|---|---|
| 3.1.1 | KPI 카드 | 4열 고정 | 1280px 미만 시 2열, 980px 미만 시 1열 |
| 3.1.2 | 차트 카드 헤더 | 제목 + 서브 | 우측에 작은 도움말(?) 아이콘 추가, 호버 시 차트 해석 가이드 툴팁 |
| 3.1.3 | 새로고침 버튼 | 일반 버튼 | 마지막 갱신 시각 표시 (예: "1분 전 갱신") |
| 3.1.4 | 내보내기 | 단일 버튼 | 드롭다운(엑셀/CSV/PDF 보고서) |
| 3.1.5 | 빈 상태 | 단순 텍스트 | 일러스트 + CTA(필터 초기화 등) |
| 3.1.6 | 테이블 행 | 클릭만 가능 | 우클릭 컨텍스트 메뉴(편집/메모/이력 보기) |

### 3.2 수금 미정 관리 페이지

- 우선순위 자동 계산 컬럼 추가 (지연일수 × 금액 가중치)
- 일괄 액션 바 (체크박스 다중 선택 → 상태 변경/담당자 변경/메모 일괄 추가)
- 알림 발송 버튼 클릭 시 모달(메일/SMS/슬랙 채널 선택)

### 3.3 로우 데이터 페이지

- 컬럼 표시/숨김 토글 메뉴 (기어 아이콘)
- 사용자별 필터 프리셋 저장/불러오기
- 스크롤 시 헤더 sticky 유지 (현재 sticky 적용되어 있으나 검수 필요)
- 페이지 하단에 합계/평균/건수 sticky 푸터

### 3.4 영업 대시보드 (리드 파이프라인)

- 칸반 카드: 정체일수 표시 위치를 우측 상단 → 좌측 컬러 바로 이동, 시각적 강조
- 정체 리드 알림 카드: 정체일 기준 정렬 + "오늘 컨택할 3건" 추천 영역
- 단계별 전환율 KPI 추가 (예: 제안서 → 견적 전환율 X%)
- 리드 상세 페이지에 **다음 액션 추천** 위젯 (스코어와 정체일 기준)

### 3.5 모바일 최적화 (필수)

- 사이드바 햄버거 → 슬라이드 인, 오버레이 추가
- KPI 카드 가로 스크롤 (모바일에서 1열 누적이 너무 길어지므로)
- 테이블 → 카드 뷰 자동 전환 (540px 미만)
- 차트 높이 자동 축소 (모바일 220px → 160px)

## 4. 신규 페이지 디자인 요청

### 4.1 사용자/권한 관리 페이지 (관리자 전용)

- 사용자 목록 테이블 (이름/이메일/소속/역할/마지막 로그인)
- 역할: 관리자/IT사업부장/사업부장/영업담당/조회전용
- 행별 권한 편집 모달 (사업부별 읽기/쓰기 체크박스 매트릭스)
- 신규 사용자 초대 버튼 → 모달

### 4.2 감사 로그 (Audit Log) 페이지

- 시간 역순 타임라인 또는 테이블 뷰
- 필터: 기간/사용자/대상 테이블/액션 종류(생성/수정/삭제)
- "이전값 → 변경값" 시각적 비교 (영업 리드 상세의 변경 이력 스타일 재사용)

### 4.3 알림 센터

- 사이드바 footer 종 아이콘 클릭 시 우측 슬라이드 패널
- 알림 카테고리 탭: 전체/지연/멘션/시스템
- 읽음/안읽음 구분, 일괄 읽음 처리

### 4.4 보고서 빌더 (간단 버전)

- 기간 + 사업부 + 차트 종류 선택 → 미리보기 → PDF 출력
- 회사 로고/제목/생성자/일자 헤더 자동 삽입
- 임원 보고용 1페이지 요약 템플릿

## 6. 접근성/품질

- 모든 인터랙티브 요소 키보드 접근 가능 (Tab 순서 검수)
- 색상만으로 상태 구분 금지 → 아이콘/텍스트 병기
- 명도 대비 WCAG AA 이상 (특히 회색 텍스트)
- focus-visible 스타일 명시 (키보드 사용자 시각 피드백)
- 스크린 리더용 aria-label 차트/버튼 일괄 정리
