4종 닫기 트리거 — 동작 검증

  • ① 햄버거 ↻ → 토글 (열기/닫기) OK
  • ② 메뉴 항목 선택 → 닫힘 + 라우팅 OK
  • ③ Scrim(어두운 영역) 탭 → 닫힘 OK
  • ④ ESC 키 → 닫힘 OK
  • ⑤ X 버튼 → 닫힘 OK

부가 동작

  • 본문 scroll-lock OK
  • aria-modal + role="dialog" OK
  • aria-expanded 동기화 OK
  • prefers-reduced-motion 폴백 OK
  • 닫힘 시 햄버거 포커스 복귀 OK
  • focus trap (사이드바 내부 순환) 데모 단순화 — 실 구현 시 추가
